岩淵勇樹「半熟語」概要

概要

半熟語は「遠隣～OnLin～」というシリーズの中の1作品です。 右脳と左脳をフル活用して熟語を推測する脳トレコンテンツとなっております。 2人で楽しむ場合は、お互いのディスプレイ同士を横に並べると答えを確認できるようになっています。

遊び方

遊び方画像

一人で楽しむ

  1. トップページにアクセス
  2. 左か右のどちらかを選び、合体した漢字を推測します。
  3. 「熟語」を選択し、答えを確認します。

二人で楽しむ

  1. 左の人は左の人メニュー、右の人は右の人メニューにアクセス
  2. お互い同じ項目を選び、お互いのパーツを推測します。
  3. 携帯電話の画面を合わせて合体漢字、完成！
