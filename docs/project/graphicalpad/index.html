概要

画像処理ソフトのような操作性を求めて、テキストベースでない新たな数式処理システムを作りたい、という理念を基にこのアプリケーションを作りました。

表計算やグラフとの連動で新たな操作感を創り出すことが目標です。 e-Learningの補助ツールとして活用できるほか、Mathematicaなどにはない操作性で研究ツールとしても実力を発揮できると思われます。

Version 2 はビジュアルプログラミング環境として鋭意制作中。
