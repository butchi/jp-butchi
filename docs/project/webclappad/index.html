概要

コンサートでの曲の演奏の後や、講演の後などに、拍手は必ず起こります。本プログラムは、リアルな拍手により近づけることを目標に作られています。

現在ある「Web拍手」と呼ばれるシステムは、mixiの「イイネ！」ボタンと同じで、ページの評価を行っているに過ぎません。

本プログラムでは、KORG社が開発しているKAOSS PADのようなインターフェースを用いてよりリアルな拍手を実現しています。

それにより、音声による新たなコミュニケーションが発生します。

動画の再生後や、ブログパーツとしての利用を想定しています。

使い方

  1. 右上の録音・再生ボタンまたは、左のパッドをクリックします。
  2. シークバーが動いている間に、左のパッドの好きな部分を好きなだけクリックします。上下は音量が異なり、左右は拍手のこもり具合が異なります。
  3. 録音終了後、再び録音・再生ボタンまたは左のパッドをクリックすると、保存した拍手が再生されます。このとき、重ねて拍手することができます。
