キュービックパズルの迷宮

概要

2004年度前期に受講した鷲山靖先生による「キュービックパズルの迷宮」で紹介されていたパズルと、講義で自作したキュービックパズル（マイ・テトブレイン）を問題として出題します。

問1

正方形でできた16×20のマスに、立方体の展開図（全部で11種類）をできるだけ多く敷き詰めてください。

52個が最高記録です。それより多く（53個）は敷き詰められないはずです。（ちなみに著者が講義の中で最高記録を出し、先生に「君は天才だ」と褒められました。2004年のゴールデンウィークはこの問題で潰れました。）
